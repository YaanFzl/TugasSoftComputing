# Soft Computing Web3

A modern, interactive web application for learning and simulating Soft Computing concepts.

## Features

-   **Genetic Algorithms**: TSP and Knapsack solvers.
-   **Fuzzy Logic**: Mamdani and Sugeno inference systems.
-   **Neural Networks**: Logic gate trainer.
-   **3D Visuals**: Interactive particle backgrounds.

## Running Locally

```bash
pip install -r requirements.txt
python main.py
```
